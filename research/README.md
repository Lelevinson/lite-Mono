# Research Artifacts

This folder stores paper-relevant research notes, experiment summaries, and artifact indexes.

Use this folder for material that may later support the paper:

- dataset-construction evidence
- calibration/label-quality decisions
- baseline model runs
- quantitative summaries
- paper table/figure candidates

Do not store large generated outputs here directly. Put generated images, NumPy files, and other bulky local artifacts under `research/generated/`, which is ignored by git.
