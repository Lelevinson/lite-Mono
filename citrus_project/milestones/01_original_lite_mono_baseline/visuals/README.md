# Milestone 1 Visuals

This folder stores milestone-local visual outputs for original Lite-Mono on Citrus.

The current panels were generated by:

```powershell
D:/Conda_Envs/lite-mono/python.exe citrus_project/milestones/01_original_lite_mono_baseline/analyze_lite_mono_citrus_results.py --split val
```

Each panel shows:

1. RGB input
2. raw predicted depth
3. median-scaled predicted depth
4. dense LiDAR label on valid pixels
5. evaluation valid mask
6. absolute error after median scaling

Current validation selection uses `median_scaled_a1`:

1. good sample: index 420, `a1=0.8264`
2. typical sample: index 82, `a1=0.4784`
3. bad sample: index 442, `a1=0.0468`

Plain meaning:

- good means most valid pixels are within the strict 25% depth threshold after median scaling
- typical is close to the split median behavior
- bad means almost no valid pixels pass the strict threshold after median scaling

The visual panels are interpretation aids. The official full-split metrics remain in `../results/`.
